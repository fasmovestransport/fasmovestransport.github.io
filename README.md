# FAS Moves Transport

This repository contains a simple static website for FAS Moves Transport that is ready to be published on GitHub Pages.

## Files
- index.html - Main landing page
- styles.css - Styling for the site
- .nojekyll - Prevents GitHub Pages from processing the site with Jekyll

## Publish on GitHub Pages
1. Push these files to the repository's main branch.
2. Open the repository on GitHub.
3. Go to Settings > Pages.
4. Select Deploy from a branch and choose the main branch with the / (root) folder.
5. Save and wait for GitHub to publish the site.
